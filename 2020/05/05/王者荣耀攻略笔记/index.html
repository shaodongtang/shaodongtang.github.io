王者荣耀攻略笔记

发表于 2020-05-05 | 更新于 2020-11-18 | 分类于 王者荣耀 | 阅读次数：

机制解析

视野机制

  1. 英雄，大致上就是如果双方都不在草丛的情况下，你的队友或者你在屏幕内可以看到的敌方英雄，会共享给你的所有队友。同样你若是出现在敌方任一英雄的视野里，你的位置也会出现在对方所有玩家的小地图中。
  2. 防御塔，向敌方防御塔靠近时，如果出现了防御塔攻击范围的黄色圆圈，那么恭喜你，你的视野已被敌方获取。此时就代表视野被防御塔捕获，而敌方英雄就可以通过小地图看到咱们的位置。
    • 注意： 防御塔的实际探测范围要比显示的黄圈大。
  3. 兵线，小兵提供视野的范围与英雄相近，大致就是你在以你操作的英雄为屏幕中心，你在屏幕上【非小地图】能看清敌方兵线全况时，你的视野也会被敌方全员知晓。另外，即使一条兵线全部死亡，仍然会存在短暂视野探照能力，这往往是你视野暴露的原因。
    • 兵线死亡以后，视野还会有短暂的残留时间。
  4. 特殊技能，比如成吉思汗的鸟，司马懿的被动，鬼谷子的大招等。这些都可以获取敌方视野，相应的以后可别在成吉思汗鸟飞过头顶，头上出现一只眼【可视标志】时还傻傻的蹲在现在的草丛里了。

防御机制

① 伤害的计算公式： $$\text{伤害}=\text{攻击值}－\text{攻击值}\times\text{减伤}$$
② 减伤的计算公式： $$\text{减伤}\%=\frac{N}{N+602}\times100\%$$
其中，$N$ 就代表护甲值或者魔抗值 $$N=\text{防御值}-\text{穿透值}$$

吸血机制

法术吸血

单体技能吸血效果为100%、中小范围技能吸血效果为50%、大范围技能吸血效果为35%。

另外关于法术吸血还有两点需要注意：

①痛苦面具、回响之杖等法术装备的被动，造成的法术伤害均为50%吸血效果。

②贪婪铭文的吸血不受制裁之刃、梦魇之牙的影响，而且还享受不死鸟之眼被动加成。

物理吸血

物理吸血的上限为100%

移速机制

移速 ≥ 575 超出575部分的增益只生效50%
500 ≤ 移速 ≤ 575 500~575区间内的移速增益只生效80%
移速 ≤ 250 低于250部分的减速只生效50%
