厨房平面布局

橱柜产品基础知识

橱柜的重要组成：地柜、吊柜、台面、高柜、电器、各类五金功能件等。

烟机常规尺寸：

  • 侧吸：最小750mm，最宽850mm
  • 欧式：895mm（中间的烟道宽度一般380-420mm）
  • 注意：烟机的深度不全都是330mm以内，有370mm以上的，若烟机深度370mm，柜体尺寸则390mm以上（不含门版）

蒸箱/烤箱/消毒柜/常规尺寸：

  • 电器尺寸：560/565mm
  • 匹配柜体宽度：600/605mm

冰箱常规尺寸：

  • 单门：600-700mm（深度600左右）
  • 双门：900-1100mm（深度700左右）
  • 内嵌冰箱柜需要在左右预留50mm以上的散热空间

橱柜的布局形式

（1）”一”字形橱柜

（2）”L”字形橱柜

（3）”U”字形橱柜：相对应的两边橱柜之间应预留位置大于或等于1200mm才方便使用

（4）岛形橱柜：

橱柜设计的注意事项

（1）橱柜设计步骤：冰箱 → 星盆柜 → 灶炉柜 → 功能柜（消毒柜或烤箱）→ 吊柜

（2）常用功能柜尺寸：

  • 水盆柜：单盆/双盆柜体的深度 ≥ 500mm（不包含门板），单盆柜体宽度为600-700mm，双盆的柜体的宽度为800-900mm。
  • 炉灶柜：单灶/双灶柜体深度 ≥ 500mm（不包含门板），单独炉灶柜宽度为600-900mm
  • 消毒柜：深度≥550mm（不包含门板），消毒柜深度为460mm左右，一后面留散热空间，二消毒柜电位也在后面，宽度基本就是600mm和605mm，烤箱，蒸箱，微波炉柜的宽度同消毒柜。
  • 拉篮柜：调味拉篮宽度为150-500mm（市面上的五金能找到），碗碟拉篮宽度尺寸为700-900mm，碗碟拉篮柜的尺寸最好是以100mm为进度，以50mm为进度的市面上相对于来说比较少。
  • 油烟机吊柜：吊柜宽度设计比客户购买的烟机宽度宽10-15mm即可，深度比烟机脖的深度深15-20mm即可，门板比柜体高50mm（留烟机吊码网上挂的位置）。

橱柜柜体拼搭

橱柜设计操作步骤

  • 地柜拼搭：柜体拼搭 → 封板 → 星盆/龙头安装 → 炉灶安装 → 台面安装 → 脚线安装
  • 吊柜拼搭：吊柜 → 封板 → 顶线安装
  • 样式材质替换：更换门板样式 → 拉手替换 → 柜体材质更换 → 门板材质更换
  • 空间装饰：智能布置 → 硬装
  • 空间渲染：灯饰 → 补光 → 外景图 → 太阳光 → 渲染效果图

地柜拼搭

（1）切角地柜

  • 三维家位置：定制 → 地柜 → 转角地柜 → 切角地柜

（2）水盆柜

  • 位置：定制 → 地柜 → 功能地柜 → 水盆地柜

  • 水盆的安装：点击水盆柜的侧板或者背板 → 电器五金 → 星盆安装 → 选择合适的水盆 → 选择

    注意：水盆的尺寸必须小于水盆柜的尺寸

  • 水龙头的安装：点击水盆 → 右键 → 电器五金 → 龙头安装 → 选择合适的龙头

（3）炉灶地柜

  • 位置：定制 → 地柜 → 功能地柜 → 炉灶地柜
  • 炉灶的安装：点击侧板或者背板 → 电器五金 → 灶台安装 → 选择合适的灶台 → 选择

（4）直角转角地柜

  • 位置：三维家位置：定制 → 地柜 → 转角地柜 → 直角转角地柜

（5）消毒柜

  • 位置：定制 → 地柜 → 功能地柜 → 电器地柜

台面的安装：

  • 右键墙体 → 全部台面 → 台面安装 → 选择合适的前挡水截面 → 选择合适的后挡水截面 → 选择材质
  • 前延伸20，若不加此20，则转角台面不会自动连接

地脚线的安装

  • 右键墙体 → 全部线条 →脚线安装 → 选择合适的型号 → 选择材质

    型号的选择：若做铝塑或者PVC，则选择MF42

  • 前延伸20

吊柜拼搭

切角吊柜i

  • 位置：定制 → 吊柜 → 转角吊柜 →切角吊柜
  • 切角预留20mm，因为柱子可能不平。
  • 离地高1500mm，方便拿东西

烟机吊柜

  • 位置：定制 → 吊柜 → 功能吊柜 →烟机吊柜
  • 欧式烟机：烟机宽度895mm+15mm，烟机柜则做910，920mm
  • 特殊参考值 → W1：若烟机钵为330mm，则做350mm
  • 特殊参考值 → H1：门板比柜体高50mm（留烟机吊码网上挂的位置）
  • 注意：炉灶与烟机吊柜中线居中

翻门吊柜

  • 位置：定制 → 吊柜 → 翻门吊柜

开门吊柜

  • 位置：定制 → 吊柜 → 开门吊柜

注意事项::

  • 转角预留50mm

智能饰品布置

饰品的布置

智能布置 → 智能饰品 → 选取喜欢的风格 → 应用 → 删除不需要的饰品

电器的安装：

  • 位置；云素材 → 公共库 → 电器 → 厨房电器
  • 缩放：C键，双击取消
  • 烟机位置：云素材 → 公共库 → 电器 → 厨房电器 → 烟机

吊顶的安装

厨房的吊顶一般为集成吊顶

  • 位置：云素材 → 公共库 → 硬装 → 吊顶 → 集成吊顶 → 选择合适的拖进去 → 右键 → 铺满
  • 按住 Ctrl 键，拖动位置

窗帘的安装

  • 位置：云素材 → 公共库 → 装饰 → 布艺软饰 → 窗帘

瓷砖的安装

横向分割？

  • 点击墙面 → 横向分割 → 点击一次墙面 → 再点击一次墙面 → 右键取消 → 拖瓷砖
  • 定制 → 硬装定制 → 铺砖 → 三维家 → 拖动到蓝色区域 → 点击其中一块砖 （一般为400×400）→ 定制（从哪里铺砖）

空间布灯渲染

补光

单空间推拉门曝光太厉害，加一面墙挡住

  • 补光：云素材 → 公共库→ 装饰 → 灯饰 → 补光
  • 若厨房吊柜下太暗了，可添加灯带

添加外景

  • 2D模式 → 云素材 → 公共库→ 门窗 → 外景

渲染

效果图 → 渲染效果图
