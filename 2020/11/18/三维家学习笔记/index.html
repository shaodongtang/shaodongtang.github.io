下单全流程操作讲解

取消隐藏：显示 → 管理显示对象 → 选中需要取消隐藏的部分 → 取消隐藏

尺寸查询：选中要移动的柜体 → 点击板件

下单

客户管理 → 添加客户 → 填写信息 → 立即提交 →

  • 客户备注：喜好？喜欢什么样的风格？什么样预算的东西？环保问题？

报价

​ 五金安装：按住 Ctrl 键框选 → 将鼠标箭头放粉色区域 → 右键 → 下单拆单 ，下单安装五金 → 选择合适的五金方案 → 确定

​ 按住 Ctrl 键框选 → 结算 → 定制报价 → 选择对应的报价方式 → 保存本地 → 报价检查 → 签合同 → 下单

  • 隐藏模型可点是，也可点否

如何检查报价？

  1. 检查单价是否有0？若存在单价为0，检查板材的材质是否正确。若材质正确依然为0，可能是用了老模型，将原模型删除重新替换即可。
    • 注：若使用软件的测试材质将不报价。
  2. 核对尺寸、数量？
  3. 小的五金不报价，因为它们包含在层板里。只有铰链单独报价
  4. 衣通模型自动生成后会默认自动携带俩模型，不报价的这俩为旧版模型，因此价格为0
  5. 门板、抽面的报价不足0.22m^2​按照0.22平米收费。
  6. 抽屉的计价方式：抽屉用平米计算，抽面用门板计算，抽芯按个数计算

下单生产？

① 生成订单号 → ② 检查

  • 生成订单号：按住 Ctrl 键框选 → 右键 → 下单拆单 → 下单，填写对应的信息 → 点击下单

    • 订单名称命名：空间+名称

    • 预约安装时间：大于一个月

    • 联系人、联系方式：针对物流信息、发货信息？不是同一人则需更换。
    • 备注：若订单成功生成，则会出现在定制类订单系统，订单管理里

  • 检查：设计是否存在问题？尺寸是否核对清楚？柜体干涉检查？

  • 柜体干涉检查：快捷键：Ctrl+1 → Ctrl+0 → Ctrl+1 → Ctrl+0

多功能房平面布局

榻榻米柜体建模

榻榻米拼搭 → 书柜拼搭 → 吊柜拼搭 → 书桌 → 收口板 → 顶封板

榻榻米的拼搭

  • 榻榻米的门板长与宽不能同时超过600mm
  • 拼搭步骤：榻榻米两侧的封板 → 放柜体 → 侧板
  • 拼接缝的处理：选中柜体 → 更多参数 → 见光方向 → 前后见光↔左右见光
  • 抽面柜深度=空白宽度-18mm（门板尺寸）

榻榻米之上的书柜：

  • 书柜深度：标准300-350mm
  • 榻榻米的假门宽度需预大书柜深度20-50mm
  • 放置：物体上
  • 标准功能件：选中柜体 → G
  • 中立板/层板居中：点击中立板/层板 → 右键 → 均布 → 输入均布数量？ → 确认
  • 掩门生成：B键进入板件模式→ 按住Ctrl键，上下左右依次选中 → 在粉色区域右键 → 门板 → 掩门生成

书桌/电脑桌

  • 书桌深度：笔记本一般为550mm，台式电脑600mm

  • 书桌上的吊柜距桌面600-750mm

  • 靠墙添加封板，板件模式更改桌面尺寸以及距墙，坐标-50

  • 桌面的倒圆：

    • 按B键进入板件模式 → 选中桌面 → 右键 → 板件点编辑 → 点击倒圆的点 → 先随便选一个类型，再点外切圆弧 → 切线长度（最小30mm） → 保存

    • 倒圆之后侧板突出的处理：更改侧板深度：原来深度减去倒圆切线长度（30mm）

材质的更换

  • 柜体材质更换 → 门板材质

空间软装装饰

  • 硬装：地板 → 墙面 → 吊顶
  • 软装：家具 → 饰品（窗品、窗帘、陈设品）

瓷砖/木地板→ 窗帘 → 墙纸 → 椅子 → 床品 → 书籍 → 地毯 → 装饰品 → 添加外景

台面结构解析

台面的结构

台面的结构主要有两种：

  1. 平板式台面：无前挡水

  2. 有前下挂台面：有前后挡水，用铝垫条垫高

    • 石英石台面组成分为：后档水、垫条、台面面板、前沿（石英石台面前沿采用45度或90度与面板粘接）、滴水槽、前沿加固条；
    • 常规前沿厚度40mm，返收60mm（包含垫条厚度），挡水45mm

常见的台面类型：石英石台面、木质台面、人造石台面、不锈钢台面、岩板台面s

CAD厨房平面布局

冰箱的放置：放厨房比较方便拿东西，放厨房门口

双门冰箱预留1000-1100mm

炉灶：炉灶宽度一般为800-900mm，若与烟机匹配宽度，则一般做900mm

水盆：一般放窗户下，见光比较好

消毒柜：宽600mm

用圆标注炉灶柜，水盆柜的中心点

CAD台面绘制

台面标注：前挡水，后挡水，接驳处，见光面，炉中，盆中，填写相关信息

注意事项：

  1. 台面见光的地方要比柜体多出30mm，
  2. 当高柜旁边有地柜的时候，要标注此台面见光30mm，否则侧边就不会做下返边
  3. 大多数的台面最大宽度是2400mm，水盆和炉灶开孔位置要离接驳线远一些
  4. 接驳线要斜角接的话计算价钱的时候重叠位也要算进去

厨房平面布局

橱柜产品基础知识

橱柜的重要组成：地柜、吊柜、台面、高柜、电器、各类五金功能件等。

烟机常规尺寸：

  • 侧吸：最小750mm，最宽850mm
  • 欧式：895mm（中间的烟道宽度一般380-420mm）
  • 注意：烟机的深度不全都是330mm以内，有370mm以上的，若烟机深度370mm，柜体尺寸则390mm以上（不含门版）

蒸箱/烤箱/消毒柜/常规尺寸：

  • 电器尺寸：560/565mm
  • 匹配柜体宽度：600/605mm

冰箱常规尺寸：

  • 单门：600-700mm（深度600左右）
  • 双门：900-1100mm（深度700左右）
  • 内嵌冰箱柜需要在左右预留50mm以上的散热空间

橱柜的布局形式

（1）”一”字形橱柜

（2）”L”字形橱柜

（3）”U”字形橱柜：相对应的两边橱柜之间应预留位置大于或等于1200mm才方便使用

（4）岛形橱柜：

橱柜设计的注意事项

（1）橱柜设计步骤：冰箱 → 星盆柜 → 灶炉柜 → 功能柜（消毒柜或烤箱）→ 吊柜

（2）常用功能柜尺寸：

  • 水盆柜：单盆/双盆柜体的深度 ≥ 500mm（不包含门板），单盆柜体宽度为600-700mm，双盆的柜体的宽度为800-900mm。
  • 炉灶柜：单灶/双灶柜体深度 ≥ 500mm（不包含门板），单独炉灶柜宽度为600-900mm
  • 消毒柜：深度≥550mm（不包含门板），消毒柜深度为460mm左右，一后面留散热空间，二消毒柜电位也在后面，宽度基本就是600mm和605mm，烤箱，蒸箱，微波炉柜的宽度同消毒柜。
  • 拉篮柜：调味拉篮宽度为150-500mm（市面上的五金能找到），碗碟拉篮宽度尺寸为700-900mm，碗碟拉篮柜的尺寸最好是以100mm为进度，以50mm为进度的市面上相对于来说比较少。
  • 油烟机吊柜：吊柜宽度设计比客户购买的烟机宽度宽10-15mm即可，深度比烟机脖的深度深15-20mm即可，门板比柜体高50mm（留烟机吊码网上挂的位置）。

  • 位置：定制 → 吊柜 → 转角吊柜 →切角吊柜

烟机吊柜

  • 位置：定制 → 吊柜 → 功能吊柜 →烟机吊柜

  • 特殊参考值 → W1：若烟机钵为330mm，则做350mm
  • 特殊参考值 → H1：门板比柜体高50mm（留烟机吊码网上挂的位置）
  • 注意：炉灶与烟机吊柜中线居中

家用酒柜设计

用餐空间酒柜设计

酒柜常用与不常用高度

  • 常用区高（730mm-2010mm）以正常姿态（直立或者俯身）便可取物，可设计存放常用物品，如酒杯、餐具、等；

  • 不常用区（80mm-730mm/2010以上）的取物比较困难，需要弯腰或者垫脚动作，可设计存放不常用物品，如珍藏的美酒等。

  • 下柜不常用区一般做800-1100mm

酒柜的基本款式

简欧酒柜

  • 设计讲究对称原则
  • 两侧加门，上玻璃门，下实心掩门
  • 中间多设计酒格以及酒杯架，下部分可以设计抽屉起到装饰性效果，或者更多元化

装饰性酒柜

  • 主要用来摆放摆件
  • 不规则设计
  • 穿插式设计红酒格、酒杯架和抽屉装饰
    效果多元化

带吧台酒柜：需用餐空间大

  • 酒柜内飘出吧台
  • 受年轻人喜爱
  • 吧台起到分割空间及临时用餐的作用

带餐桌式酒柜：适合小空间

  • 客户一站式购物避免挑选餐桌的麻烦
  • 餐桌酒柜风格一体化，美观大方
  • 有利于增大单值

上薄下厚式酒柜

  • 酒柜设计空间足够大

  • 上部分设计为酒柜展示柜

  • 下部分为独立的较厚掩门储物柜，这样突出位置可以放榨汁机咖啡机等，方便制作美食

结合冰箱式酒柜

  • 将冰箱融入到酒柜的设计里头，一体化美观
  • 设计时可把下柜厚度与冰箱呼应齐平，这样冰箱不突兀，下柜还有可操作的平台

结合冰箱式酒柜
将冰箱融入到酒柜的设计里头，一体化美观
设计时可把下柜厚度与冰箱呼应齐平，这样冰箱不突兀，下柜还有可操作的平台

隔断式酒柜：装饰、储物融为一体

  • 隔断酒柜既有展示柜的效果，也起到分隔空间的作用。
  • 整体美观大气，充分体现定制的优势，适合用在大户型
  • 将两个空间合理连贯起来

L形酒柜鞋柜一体式

  • 适合入户到餐厅有拐角墙体的户型
  • 一体化设计将入户柜和酒柜连贯一体，充分体现定制优势，将阴阳角完美隐藏，既美观又实用

色彩搭配的黄金法则
