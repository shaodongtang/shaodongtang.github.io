CAD教学

直线

基本命令

  1. 范围选中：左键点击它的右下方某处 → 左键点击它的左上方某处
  2. 查找已画的图形？Z+空格 → E+空格

绘图命令：

  1. 直线：L+空格 → 输入长度 → 左键
  2. 矩形：REC → 选一个点 → 再选一个点
    • REC+空格 → 矩形的顶点 → D+空格 → 输入长度+空格 → 输入宽度+空格 → 左键
    • REC+空格 → 矩形的顶点 → @长度,宽度 空格
      注意：使用@指令矩形确定的顶点是最下方的顶点。若想确定的顶点在矩形的左上角，则需要用 负的宽度。
  3. 多行文字：T+空格 → 拉一文本框 → 输入文字 → 调节大小
  4. 注释箭头：LE+空格 → 左键(箭头) → 左键(线端)
  5. 两条直线延长相交：F+空格 → 选中其中 → 条线 → 光标移动到另外 → 条线 → 左键
  6. 两条直线弧形延长：F+空格 → 选中其中一条线 → R+空格 → 输入半径 → 选中另一条线→ 空格
  7. 延伸：选中两条线 → EX+空格 → 光标移动到要延升的线 → 左键
  8. 图案填充：选中需要填充的区域 → H+空格 → 选择填充的样式 → 左键
  9. 正多边形：POL+空格 → 输入多边形的边数 → 左键指定多边形的中心点或者输入边长

修改命令：

  1. 删除：
    • E+空格 → 点击需要删除的线 → 空格
    • 点击需要删除的线 → E+空格
  2. 等分：DIV+空格 → 选中要等分的线 → 输入几等分？ → 连线 → 删除标记
  3. 复制：
    • 选中复制对象 → CO+空格 → 移动找到对象✘处 → 点击拉到需要的位置 → 左键
    • 选中复制对象 → Ctrl+C → Ctrl+V
  4. 镜像：选中对象 → MI+空格 → 选择对称轴 → 左键点击对称轴 → 空格，选择否(不删除)
  5. 偏移：O+空格 → 输入偏移长度 → 选中要偏移的线 → 选择偏移的位置即可
  6. 移动：
    • 选中要移动的对象 → M+空格 → 移动找到对象✘处点击拉到需要的位置 → 左键
    • 固定移动距离：选中要移动的对象 → M+空格 → 选点(左键) → 输入移动距离 → 确认(左键)
  7. 旋转：选中需要旋转的对象 → RO+空格 → 选圆心 → 选择旋转的位置
  8. 缩放：选择对象 → SC+空格 → 选一点 → 输入倍数 → 空格
  9. 修剪：TR+空格+空格 → 点击需要删除的线

标注命令

  1. 线性标注：QD+空格 → 选中至少两条平行的线 → 右键 → 左键
    • 标宽度？选中竖线 → 右键(图的上、下)
    • 标高度？选中横线 → 右键(图的左、右)
  2. 对齐标注：DAL+空格 → 选中两个端点 → 左键
  3. 角度标注：DAN+空格 → 选中其中一条线 → 选中另一条线 → 左键
