摘要：营销与公关案例分析

关键词：市场营销、公关、粉圈化

市场营销

小米的营销分析

小米的营销分析：造神、粉圈化、画饼、“性价比”、站在道德制高点、蹭热度、参与感

参与感：微博的话题讨论和抽奖

案例：花200万由由世界级著名大师改新logo，但由于样式看着简单，米粉称雷军被骗了。

分析：人们最喜欢看“傻子”，拉别人一起看傻子

  • 新logo样式看着简单 → 米粉称雷军被骗了 → 热度爆炸 → 反转：改新logo需绕过许多种设计专利，大厂改新logo确实费钱 → 雷军没有被骗，在大气层 → 雷军牛逼

最终效果：200w获得了千万级的营销效果：#小米新logo#的话题在微博的阅读量已达2.7亿，讨论超3.7万

占据道德制高点： “良心”、“厚道”、“性价比”、“朋友”、“动了整个行业蛋糕”、“智商优越感”

  • 造“神”运动：雷军本人极具有励志性人设，同马斯克的营销。
  • “粉圈化”：“你可以黑小米、但不可以黑雷军”
  • 受害者立场：“n代海军”、“我们被友商黑惨了”、“生死看淡、不服就干”、“百花齐放才是春”
  • 智商优越感：“厂妹”、“智商机”、等等

最终效果：虽然小米销量不是多好，但是互联网的声量是最大的

  • 说小米不好=你收了黑钱
  • 我小米是为消费者考虑的，我“良心”、“厚道” 因此，骂小米=跟消费者作对

造谣带节奏：暗示友商偷工减料、当然自己也这么干，甚至不如友商，但造谣容易、辟谣，而且人们更愿意相信自己看到的

12月13日，小米@Redmi潘达率先发难，炮轰荣耀智慧屏在扬声器方面涉嫌“虚假宣传”。随后，卢伟冰转发助推：“我也以为是60W,让Redmi电视产品经理去研究学习。”
12月14日， 荣耀@申开朗Sky回应:确认智慧屏采用6个10W扬声器，并嘲讽小米祖传2*8W扬声器，“坐井观天”。
12月14日，某自媒体：“荣耀小米你们别吵了，这么简单的问题,拆个机不就完了吗?”

  • “喇叭数量x单个喇叭额定功率”是行业通用的扬声器参数标注标准
  • 站在道德制高点：将“6*10W”引导暗示为“60W” → 暗示虚假宣传

本意想造谣污名化友商，带节奏暗示友商虚假宣传，结果自己翻车了。

  • 造谣比辟谣简单太多，造谣比辟谣更有效
  • 这波操作是反面教材

造谣带节奏的目的：虽然自己不咋滴，但是把你搞臭，用户就得在一个垃圾和一个更垃圾中选择

分析小米的“画饼”技术

销售画饼：“耍猴”、“饥饿营销”

小米的发布会产品价格低，但需要抢购

  • 站在道德制高点：看似“良心厚道”，实际上抢不到，而且“抢不到”可以把锅甩给用户，抢到了3-5周发货
  • 以时间分摊成本：硬件成本随着时间的拉长会降低，于是慢慢放货
  • 写脚本、养“黄牛”赚钱：抢不到加价买，加的价格自然与黄牛平摊

苹果取消充电头：宣称“不环保”，以此抬价！

  • 小米并没有苹果的底蕴，直接告诉米粉“取消充电器”无疑于是作死的行为，但小米取消充电器确一片叫好
  • 小米此时依然在冲高端
  • 造势：“不会有人认为充电器是免费送的吧”
  • 发布：升级配置手机价格不变且充电器可以选择并不加价

分析：充电器以环保理由不要但不减去充电器的价格，赚到“环保用户”的充电器钱，还站在“良心”的制高点。

没有技术怎么画？炒冷饭、造概念

  • 一块钢板的艺术之旅“奥氏体304不锈钢”
  • 小米mix：造词“全面屏”，实际上夏普很早就有这设计，设计没有大厂抄是最打击人的，后来，米8抄了苹果x
  • note4x初音版，索尼
  • 隐私保护：华为n年前的技术

➀ 澎湃S1芯片的画饼之术

  • 偷换概念：我买的=我研发，澎湃S1芯片为大价钱从大唐联芯买来的方案基于LC1860的马甲版
  • 技术贬低：“芯片沙子价”，自己搞不定就贬低，让粉丝认为芯片没难度
  • 鱼目混珠：米11发布会的isp芯片，让粉丝认为澎湃芯片成了
    • isp芯片是基础芯片，技术与soc技术有可比性吗？
    • 澎湃S1是买的技术，这isp是自己研发还是买的技术？

小米isp芯片画的饼：“澎湃芯片没有放弃”、“芯片小米能做”。

小米的芯片团队独立为南京大鱼半导体和松果电子

  • 搜大鱼一样招聘信息极为罕见，包括松果的历史招聘信息，都是寥寥无几。
  • 大鱼：小米25％持股
  • 宣传“澎湃没有放弃”
    • 但是芯片部门只有在招isp的人，而“芯片”soc的团队根本不招人，无论社招还是校招，也无猎头挖人；
    • 友商OPPO也没怎么宣扬做芯片，结果模拟数字都招人 ，设计和验证都有
  • 让粉丝感觉“芯片小米能做”
    • 没有放弃是多久能有成果？能做是多久能做？

为何要画：“澎湃芯片没有放弃”、“芯片小米能做”的这个饼？

  • 冲高端没有技术支持，用户是不买账的
  • 小米确实感觉到了用户开始支持国产了

➁ 科技碰瓷画饼艺术

  • 小米与“xx”合作研发的画饼之术：三星合作研发摄像头、高通合作研发

造概念画饼

  • 华为10倍光学变焦，小米做不出来，供应链也没有，怎么办？宣传“十倍等效光学变焦”

折叠屏画饼：华为matex在MWC获最高奖

  • 雷军微博转“米？获得野鸡外媒最佳奖”
  • 微博发布“实验室折叠屏”：表示自己能做
  • 玻璃柜展示机“mixα”，不知道能不能开机，无论实用性还是难度差距华为不是一点点
  • 发布mix，然而软件适配没有做、电脑xx功能是“未来”加入（发布会小字提醒）

碰瓷高科技：让米粉认为小米也有这技术，但小米的研发占比接近于国家定义科技公司最低研发比（减税）

“性价比” 营销

米式“性价比”：研发“性价比”、工资“性价比”、招人“性价比”、设计“性价比”

事实上，性价比是个伪概念。看上去便宜的东西往往是最贵的

研发“性价比”

  • 米重金聘请芯片行业的一些资深工程师？不存在的
  • 小米科技哪来的？供应链 “首发xxx”、如果友商也有？那小米的就是“满血版”

工资“性价比”：行业低

招人“性价比”：“芯片”部门根本不招人

设计“性价比”：“没有设计就是最好的设计”、“借鉴”

  • 米8被誉为最像“苹果x”的手机

从效果来看，对于一个没有任何技术的厂商，一开始的破局之法就是“性价比”策略

  • 站在道德制高点
  • 能存活下去，虽然利润不可能“科技”厂商相提并论

但缺点就是以“性价比”吸引而来的用户不稳定、再冲高端就很难！

规避虚假宣传

极为不明显、不显眼的小字：擦边球

  • 告知义务、出了事就说是你没看清
  • 制造信息不对称误导人

利用 “小字” 规避虚假宣传的范例精选

  • Redmi 10x：“图片仅作展示，非Redmi 10x实拍”
  • 小米8：“数据来源于小米实验室，展示结构图片，均为功能示意图，并非绝对实际结构，最终以实物为准”
  • 小米？：“双5G功h预计7月份业持,法后脚OTA.界面仅为示意,以实际为淮”
  • 小米电视：“*8K视频内容指支持USB通道和在线播放24帧部分视频格式”

拼多多的营销分析

现金大转盘

“现金大转盘” → “拼多多送你现金红包” → 开点一个必为手气最佳 → 获得300-600不等的现金红包 → 点击提现 → 差多少提现？1毛 → 大转盘抽 → 抽一次1分钱 → 差3分钱提现，但抽奖次数已用完 → 分享好友可获得抽奖次数 → 差一分钱 → 抽金币，10金币=1分钱 → 差一金币 → 抽0.1个金币 → 抽到9.9个金币 → 抽0.01金币 → 差0.01金币 → 祝你福气满满 → 结束！

  • 诱人的红包数字 → 吸引用户点击
  • 抽一次就能获得299.9块的现金 → 留住用户继续抽
  • 时刻提醒只差一点点就能提现了 → 激励用户帮拼多多完成拉新人的任务，被拉者同样只差一点点就能提现

分析：幸存者偏差、贪小便宜、可得性、求而不得、赌徒心理

  • 幸存者偏差：随便一抽就是最佳手气，营造用户运气的“天选之子”吸引用户尝试！
    • 确实存在用户提现成功的案例，但这概率？
  • 贪小便宜：白嫖的吸引性，送你你要不要？
  • 可得性：永远只差一点
  • 求而不得：绝望中的希望
  • 赌徒心理：赌气、不服输

黑化：通过qq群下载app、xx帮砍

  • 互利帮：下载app免费送10刀 → 需要充值才能发布任务 → 充钱发现只有一人接单 → 重新发布需充值会员，非会员只能发布一条 →
    • 提现：金币与现金的比例 → 刚充值的用户要么放弃充的钱，要么冲会员及时止损

砍价免费拿

  • “免费拿” → 吸引用户点击
  • 砍一次就能砍掉90% → 留住用户继续抽
  • 眼花缭乱的特效 → 暗示用户自己今天是全宇宙最幸运的
  • 时刻提醒只差一点点就能提现了 → 激励用户帮拼多多完成拉新人的任务，被拉者同样只差一点点就能提现
